# 表单输入的数据双向绑定

# 基础用法

数据的双向绑定 -> v-model="数据来源" (data) 本质上是语法糖 负责监听用户的输入事件来更新数据

使用元素：input textarea select checkbox radio

> **<font style="color:#52C41A;">提示</font>**
>
> v-model 会忽略所有表单元素的 value、checked、selected

## 文本（Text）

input + value 语法糖

```javascript
import { createApp } from "vue/dist/vue.esm-bundler";
// import App from './App.vue'

const App = {
  template: `
    <div>
      <p>{{ inputText }}</p>
      v-model:<input type="text" v-model="inputText"/>
			input+value:<input type="text" @input="setInputValue" :value="inputText">  //语法糖
    </div>
  `,
  data() {
    return {
      inputText: "inputText",
    };
  },
  methods: {
    setInputValue(e) {
      this.inputText = e.target.value;
    },
  },
};

createApp(App).mount("#app");
```

## 选择框（Select）

change + value 语法糖

> **<font style="color:#52C41A;">注意</font>**
>
> 如果 v-model 表达式的初始值未能匹配任何配置项，select 元素会被渲染成“未选中”状态。在ios中，会使用户无法选择第一个选择项。因为这种情况下 ios 不会触发 change 事件。所以通常在第一项会添加一个 value为空传的 请选择 options的禁用选项

```javascript
import { createApp } from "vue/dist/vue.esm-bundler";
// import App from './App.vue'

const App = {
  template: `
    <div>
      <p>{{ selectValue }}</p>
      <select v-model="selectValue">
        <option value="China">中国</option>
        <option value="Russia">俄罗斯</option>
        <option value="America">美国</option>
      </select>
      <select @change="changeValue" :value="selectValue">   //change + value 语法糖
      <option value="China">中国</option>
      <option value="Russia">俄罗斯</option>
      <option value="America">美国</option>
    </select>
    </div>
  `,
  data() {
    return {
      inputText: "inputText",
      selectValue: "China",
    };
  },
  methods: {
    changeValue(e) {
      this.selectValue = e.target.value;
    },
  },
};

createApp(App).mount("#app");
```

## 复选框（Checkbox）

change + checked 语法糖

单个复选框

```javascript
<input type="checkbox" v-model="checked" />
<input type="checkbox" @change="changeCheck" :value="checked">
<label for="checkbox">{{ checked }}</label>
```

多个复选框

```javascript
{{ selectedCountries }}   // []
中国： <input type="checkbox" value="China" v-model="selectedCountries">
俄罗斯：<input type="checkbox" value="Russia" v-model="selectedCountries">
美国：<input type="checkbox" value="America" v-model="selectedCountries">
```

## 单选框（Radio）

```javascript
{{ gender }}   // ''
男：<input type="radio" value="male" v-model="gender">
女：<input type="radio" value="female" v-model="gender">
```

# 值绑定

## 复选框（Checkbox）

```javascript
{{ isAdult }}
<input type="checkbox" v-model="isAdult" true-value="成年" false-value="未成年">
```

## 单选框（Radio）

```javascript
{{ isAgreeValue }}
<input type="radio" v-model="isAgree" :value="isAgreeValue">
```

# 修饰符

## .lazy

```javascript
{{ inputText }}
<!-- input + value: 每次输入完成时，数据发生改变 -->
<input type="text" v-model="inputText">
<!-- change + value: 每次失去焦点时，数据改变 -->
<input type="text" v-model.lazy="inputText">
```

## .number

希望将输入的值转为数值类型，可以给 v-mode 添加 .number 修饰符

```javascript
<input type="number" v-model.number="num">
```

> 即使 type="number" HTML 元素的值也总是会返回字符串，如果这个值无法被parseFloat()解析，则会返回原始的值

## .trim

去除首尾的空格 给v-mode添加 .trim 修饰符

```javascript
<input type="text" v-model.trim="msg">
```
