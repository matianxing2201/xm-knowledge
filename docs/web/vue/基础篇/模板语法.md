# 模板语法

Vue的模板都是基于HTML，模板中直接写HTML都是能够被HTML解析器解析的。

Vue提供一套模板编译系统。开发者写的template -> 分析 HTML字符串 ->AST树 -> 表达式/自定义属性/指令 -> 虚拟DOM树 -> 解析真实DOM -> render

## 插值表达式

插值 -> JS表达式 不是语句 不是函数 不是赋值 声明 不是模块 只能绑定一个表达式

```javascript
const App = {
  data() {
    return {
      title: "title1",
      a: 1,
      b: 2,
      subTitle: "subTitle",
    };
  },
  template: `
        <div>
            <h1>{{ title }}</h1>   // 插值表达式
						<!--数学运算表达式-->
            <h1 :title="a + b">{{ a + b }}</h1>
            <!-- 字符串拼接 -->
            <h2>{{ 'a + b =' + (a + b) }} </h2>
					 	<!-- 判断表达式 -->
            <h3>{{ a + b > 5 ? '大于5' : '小于5' }}</h3>
            <h3>{{  title || subTitle }}</h3>
						<h3>{{  title && subTitle }}</h3>
						<!-- 使用JS Api -->
            <h4>{{ title.replace('title','') }}</h4>
            <h4>{{ subTitle.split('').reverse().join('-') }}</h4>
 						<!-- 绑定多个表达式 会报错 只能绑定一个 -->
            <h5>
                {{ 
                    'a + b =' + (a + b) 
                    title
                }}
            </h5>
						<!-- 绑定语句  不可以 -->
						{{ var a = 1 }}    声明赋值语句
						{{ a = 1 }}   赋值表达式   可以
        </div>
    `,
};
createApp(App).mount("#app");
```

## directive 指令

所有在Vue中，模板上属性的v-\* 都是指令。

Vue提供了大量的内置指令` v-if  v-else v-else-if  v-for  v-show  v-html`

开发者也可以给Vue扩展指令 ->自定义指令 v-取名

### v-once

1. 一次插值，永不更新 通常不建议使用。
2. 会影响整个标签

```javascript
给H1添加 v-once指令后 无法修改属性。
const App = {
    data() {
        return {
            title: 'title1',

        }
    },
    template: `
        <div>
            <h1 v-once>{{ title }}</h1>
            <button @click='changeTitle'>修改</button>
        </div>
    `,
    methods: {
        changeTitle() {
            this.title = 'change TITLE'
        }
    }
}
createApp(App).mount('#app');

```

### v-html

1. 插值不会解析html，因为插值是JS表达式，没有对DOM的操作 rowHTML。
2. 不要试图使用v-html做字模板 ，vue本身有一个底层的模板编译系统而不是直接使用字符串渲染。字模板放到子组件中，让模板的重用和组合更强。
3. 不要把用户提供的内容作为v-html的插值，这种插值容易导致XSS的攻击
   1. v-html动态的渲染HTML，使用基本是innerHTML
   2. innrHTML有时候会容易导致XSS攻击

```javascript
let isShow = true;
const App = {
    data() {
        return {
            title: '<h1 v-if = "isShow">this is title</h1>',    .//会把<h1></h1>也显示出来
        		//v-if 不会编译
        }
    },
    template: `
        <div>
            {{ title }}
						<span v-html="title"></span>    //会被渲染成H1的字符串
        </div>
    `,
}
```

### v-if v-else-if v-else

删除节点

通过添加<!--v-if-->注释节点来替换模板

```javascript
const App = {
  data() {
    return {
      linkIndex: 0,
      urls: [
        "https://www.taobao.com",
        "https://www.tmail.com",
        "https://jd.com",
      ],
    };
  },
  template: `
        <div>
            <div>
                <p v-if="linkIndex === 0">
                    <a :href='urls[0]'>淘宝</a>
                </p>
                <p v-else-if ="linkIndex === 1">
                    <a :href='urls[1]'>天猫</a>
                </p>
               
            </div>
            <div>
                <button @click='changeIndex(0)'>taobao</button>
                <button @click='changeIndex(1)'>tmail</button>
                <button @click='changeIndex(2)'>jd</button>
            </div>
        </div>
    `,
  methods: {
    changeIndex(index) {
      this.linkIndex = index;
    },
  },
};
```

### v-show

隐藏节点

```javascript
const App = {
  data() {
    return {
      linkIndex: 0,
      urls: [
        "https://www.taobao.com",
        "https://www.tmail.com",

            <div>
                <button @click='changeIndex(0)'>taobao</button>
                <button @click='changeIndex(1)'>tmail</button>
                <button @click='changeIndex(2)'>jd</button>
            </div>
        </div>
    `,
  methods: {
    changeIndex(index) {
      this.linkIndex = index;
    },
  },
};
```

### v-bind

参数 -> 插入表达式 v-bind:href='...' => href='https://www.baidu.com'

v-bind -> :

1. 动态的属性名参数不能出现空格和引号，HTML的合法属性名不能出现空格引号。
2. null作为属性是无效的，可以利用null解除属性绑定
3. 不能访问全局变量。

```javascript
var str = "global main";
const App = {
  data() {
    return {
      attr: "data-tag",
      tag: "h1",
      myAttr: "data-attr",
      title: "myTitle",
    };
  },
  template: `
        <div>
            <div>
                <h1 v-bind:[attr]='tag'>{{ title }}</h1>
 								<h1 v-bind:[myAttr]='tag'>{{ title }}</h1>
            </div>
        </div>
    `,
  methods: {
    removeAttr(index) {
      this.myAttr = null;
    },
  },
};
```
