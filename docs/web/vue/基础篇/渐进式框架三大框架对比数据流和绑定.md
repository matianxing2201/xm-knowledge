vue对自己框架和其他框架对比后，生产的一个特定的名词。

# 2、三大框架对比

| **Vue**                                                                                                                                  | **React**                                                                                                                           | **Angular**                                                              |
| ---------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| 1. 用户界面 View 视图层 -> 怎么把数据渲染到视图中<br/>2. 一个点 视图 -> 核心库<br/>3. Vuex o（选择集成）<br/>4. Vue-router o（选择集成） | 1. 用户界面 View 试图层 -> 怎么把数据渲染到试图中<br/>2. 一个点 视图 -> 库<br/>3. 状态中央管理（Redux）<br/>4. 路由（react-router） | 1. 综合性框架开发平台<br/>2. 项目应用 / X 视图渲染/状态的管理 - 大型应用 |

# 3、数据绑定和数据流

## 数据绑定

数据与视图渲染直接的关系

### React

单向数据绑定 -> `event`-> `state` 更改 -> 视图变更

### Vue

双向数据绑定 -> `event`-> `state/data` 更改 -> 视图变更 -> `v-modal` -> 视图变化 -> `state/data` 变更。

## 数据流

数据流淌的方向 -> 父子组件中 数据按照什么方向流动

### 单项数据流

**props:** immutable value 不可变的值

**state/data: **mutable value 可变值

| <font style="color:#404040;">React/Vue</font> | 父组件 -> state -> 子组件 -> props                                                 |
| --------------------------------------------- | ---------------------------------------------------------------------------------- |
|                                               | 子组件 -> props变更 -> 父组件 -> state 变更 <font style="color:#F5222D;">×</font>  |
|                                               | 父组件 -> props变更 -> 子组件 -> state 变更 <font style="color:#52C41A;"> o</font> |
