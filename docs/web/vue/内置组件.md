# 内置组件

# 1、keep-alive

## Props

- include - string | RegExp | Array -> 只有名称匹配的组件会被缓存 <font style="color:#FA541C;">包含</font>
- exclude - string | RegExp | Array -> 任何名称匹配的组件都不会被缓存 <font style="color:#FA541C;">排除</font>
- max - number | string -> 最多可以缓存多少组件实例

## 用法

- `<keep-alive>` 包裹动态组建时，会缓存不活动的组件实例，而不是销毁它们。
- `<keep-alive>` 是一个抽象组件，它自身不会渲染一个DOM元素，也不会出现在组件和父附件链中
- 当组件在 `<keep-alive>` 内被切换时，他的 `mounnted` 和 `unmounted` 生命周期钩子不会被调用，取而代之的是 `activated` 和 `deactivated`。
