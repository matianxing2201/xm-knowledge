# 模板引用

# ref (reference 引用)

- 引用DOM节点、引用组件实例
- 组件在挂在完毕（mounted）之后才能获取ref
- $refs本身不是响应式，所以不要再模板中使用
- 不要再计算属性中访问
- 不要尝试更改ref，ref提供获取DOM节点或者组件实例引用的

## 注意事项

ref不可以乱用，Vue本身是数据绑定机制，DOM的增<font style="color:#000000;"></font>删改查是Vue的驱动ViewModel去做的。主动的获取，或者主动的操作就有必要去思考哪里有问题。有一些情况是需要获取，但不是一定要去获取。肯定会有其他的解决方案能够替代ref

> **<font style="color:#000000;background-color:#FADB14;">WARNING</font>**
>
> <font style="color:rgb(71, 101, 130);">$refs</font><font style="color:rgb(34, 34, 34);"> 只会在组件渲染完成之后生效。这仅作为一个用于直接操作子元素的“逃生舱”——你应该避免在模板或计算属性中访问 </font><font style="color:rgb(71, 101, 130);">$refs</font><font style="color:rgb(34, 34, 34);">。</font>

## ref在DOM上的绑定

```javascript
<template>
  <div>
    <p>
      username: <input type="text" v-model="username" ref="username" />
    </p>
    <p>
      password: <input type="text" />
    </p>
  </div>
</template>;
export default {
  name: "Ref",
  data() {
    return {
      username: "Mtx",
    };
  },
  mounted() {
    console.log(this.$refs.username.value);
  },
};
```
