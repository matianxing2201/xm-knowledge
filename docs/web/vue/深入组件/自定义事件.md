# 自定义事件

## 1、事件名

通常在子组件中触发事件使用 camelCase（驼峰式）命名，父组件中添加一个 kebab-case（短横线命名）的监听器。

## 2、定义自定义事件

在Vue3中可以通过`emits`选项在组件上定义发出的事件。在Vue2中如果是定义原生的事件需要通过.native修饰符实现。

## 3、验证抛出的事件

与props的类型验证相似，如果使用对象语法而不是数组语法发出的事件，可以对他进行验证。对事件分配一个函数，函数接收传递给`$emit`调用的参数，返回一个布尔值提示事件是否有效。只会提示一个告警 对开发者适用。

## 4、`v-model`参数

默认情况下，组件上的v-model 使用属性和`update:value`作为事件，通过`v-model`传递参数来修改。

```javascript
<my-component  v-model:title='name'/>

子组件
<input type='text' :value='title' @input="$emit('update:title',$event.target.value)"/>
export default {
  name:'',
  props:{
    title:String
  },
  emits:['update:title']
}

```
