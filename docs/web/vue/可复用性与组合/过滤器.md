# 过滤器

## 简介

Vue.js 允许使用自定义过滤器，再次之前需要通过计算属性对数据包裹一层，在进行处理这样并不优雅。过滤器可用于一些常见的文本格式化。**双花括号插值和**`**v-bind**`**表达式**。过滤器应该呗添加再JavaScript表达式的尾部，由"管道"符号表示。

```vue
<!-- 在双括号中 -->
{{ itemStr | replaceNumToChs }}

<!-- 在 `v-bind` 中 -->
<div v-bind:item="itemStr | replaceNumToChs"></div>
```

## 注册方式

### 局部注册

```javascript
import { replaceNumToChs } from "@/filters";

export default {
  name: "TabItem",
  filters: {
    replaceNumToChs,
  },
};
```

```javascript
filter - index.js;
import replaceNumToChs from "./replaceNumToChs";
export { replaceNumToChs };

filter - replaceNumToChs.js;
export default (value) => {
  return value.replace(/\d+/g, (node, key) => {
    return numReplace(node);
  });
};

function numReplace(num) {
  switch (num) {
    case "1":
      return "一";
    case "2":
      return "二";
    case "3":
      return "三";
    case "4":
      return "四";
  }
}
```

### 全局注册

在创建`Vue`实例之前全局定义过滤器

```javascript
Vue.filter("replaceNumToChs", function () {
  return value.replace(/\d+/g, (node, key) => {
    return numReplace(node);
  });
});
```

## 注意点

- 当全局过滤器和局部过滤器重名时，会采用局部过滤器。
- 过滤器函数总接受表达式的值作为第一个参数（可链式操作）。<code v-pre>{{ message | filterA | filterB }}</code>
- 过滤器时JavaScript函数，所以可以接收参数 <code v-pre>{{ message | filterA('arg1', 'arg2') }}</code>
